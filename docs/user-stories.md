# User Stories — Personal Brand + Learning Hub

## Persona principal
**Jingxiao**, Product Owner freelance, utilise ce site comme vitrine professionnelle et outil de montée en compétences personnel.

---

## Page d'accueil

**US-01** — Présentation du positionnement  
As a visiteur (recruteur / client potentiel),  
I want to comprendre en quelques secondes qui est Jingxiao et ce qu'elle fait,  
So that je sache rapidement si son profil correspond à mon besoin.

**US-02** — Accès au contact  
As a client potentiel,  
I want to trouver un moyen simple de la contacter,  
So that je puisse initier une conversation sans friction.

**US-03** — Découverte du Learning Hub  
As a visiteur curieux,  
I want to explorer son Learning Hub depuis la page d'accueil,  
So that je comprenne sa démarche de développement professionnel.

---

## Sélection des Hubs

**US-04** — Vue d'ensemble des Hubs  
As a Jingxiao,  
I want to voir tous mes Hubs disponibles sur une seule page,  
So that je puisse choisir rapidement sur lequel travailler.

**US-05** — Progression visible d'un coup d'œil  
As a Jingxiao,  
I want to voir le pourcentage de compétences maîtrisées pour chaque Hub,  
So that je mesure ma progression globale sans entrer dans le détail.

**US-06** — Placeholder pour futurs Hubs  
As a Jingxiao,  
I want to voir les Hubs en préparation avec un badge "Bientôt",  
So that je garde une vision de ma feuille de route de compétences.

---

## Hub Détaillé

**US-07** — Navigation par piliers  
As a Jingxiao,  
I want to voir mes compétences organisées en 3 piliers distincts,  
So that j'aie une vue structurée et non une liste plate.

**US-08** — Suivi de statut par compétence  
As a Jingxiao,  
I want to passer une compétence de "Non commencé" → "En cours" → "Maîtrisé" en un clic,  
So that je tienne à jour mon niveau sans formulaire complexe.

**US-09** — Persistance de la progression  
As a Jingxiao,  
I want to retrouver ma progression exacte à chaque visite,  
So que je n'aie pas à tout re-saisir.

**US-10** — Détail d'une compétence  
As a Jingxiao,  
I want to cliquer sur une compétence pour voir sa description et les ressources recommandées,  
So that j'aie tout le contexte pour planifier mon apprentissage.

**US-11** — Notes personnelles  
As a Jingxiao,  
I want to écrire des notes libres sous chaque compétence,  
So that je puisse consigner des liens, réflexions ou jalons personnels.

**US-12** — Barre de progression globale  
As a Jingxiao,  
I want to voir ma progression globale et par pilier en haut du Hub,  
So que j'aie une vue synthétique dès l'ouverture de la page.

**US-13** — Distinction essentiel / bonus  
As a Jingxiao,  
I want de voir quelles compétences sont "Essentielles" vs "Bonus",  
So que je priorise les bonnes compétences en premier.

---

## Extensibilité

**US-14** — Ajout d'un nouveau Hub sans toucher à l'interface  
As a Jingxiao (ou un développeur),  
I want to créer un nouveau Hub en ajoutant uniquement un fichier de données,  
So que la plateforme soit maintenable sans refactoring.
